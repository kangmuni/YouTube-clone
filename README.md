# YouTube-Clone / 반응형 유튜브 클론
> **스택**
1. HTML
2. CSS
3. JavaScript

> **기술**
1. HTML 태그들을 이용해 복잡할 수 있는 레이아웃을 시멘틱적 구조로 잡기
2. 반응형으로 가로 사이즈가 줄면 정렬이 세로로 바뀜
3. 반응형으로 가로 사이즈가 커지면 검색창이 커짐
4. 동영상 설명란에 '간략히' '자세히' 토글버튼 생성
